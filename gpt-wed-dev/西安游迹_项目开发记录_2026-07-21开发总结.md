# 四、特殊地点系统

## 背景

西安奥体中心不是旅游景点，不应该进入：

-   places.js
-   景点详情页
-   周边探索体系

因此建立独立地图点体系。

新增：

    miniprogram/data/map-points.js

加入：

    西安奥体中心

    地址：
    陕西省西安市灞桥区奥体大道2020号

    经度：
    109.032531

    纬度：
    34.378704

设计：

-   只显示地图
-   不进入景点详情
-   不参与周边景点搜索

首页和 full-map 显示：

-   5个旅游景点 marker
-   1个特殊地点 marker

特殊 marker：

    ID = 1000

点击不跳转。

# 五、景点周边探索功能

新增：

    utils/map-utils.js

功能：

-   坐标有效性判断
-   Haversine距离计算
-   周边景点搜索

升级：

`place-map`

从：

    当前景点一个marker

升级为：

    当前景点 ⭐

    +

    附近景点 ●

规则：

-   半径：3公里
-   最多显示5个附近景点
-   按距离排序
-   当前景点 marker ID = 1
-   附近景点从 ID = 2 开始

增加：

`include-points`

实现地图自动调整视野。
