# 西安游迹（XI'AN TRAVEL GUIDE）

## 项目介绍

西安游迹是一个微信小程序旅行助手，以西安地图为入口，帮助用户按区域查看景点与美食，并提供开放时间、门票、游览建议和参考位置等旅行信息。

## 当前功能

- 首页西安地图
- 精选景点 marker
- 用户当前位置定位
- 区域攻略
- 景点详情
- 景点开放时间
- 景点门票信息
- 建议游览时长
- 景点注意事项
- 景点参考位置地图
- 美食分类展示
- 美食详情
- 美食参考位置地图

## 技术架构

- 微信小程序原生框架
- 微信原生 `map` 组件
- 本地 JavaScript 数据驱动
- Git + GitHub 版本管理

## 页面结构

```text
miniprogram/pages/
├── home/          首页地图与精选景点
├── region/        区域攻略
├── detail/        景点详情
├── food/          美食分类展示
├── food-detail/   美食详情
├── food-map/      美食参考位置地图
└── place-map/     景点参考位置地图
```

## 数据结构

项目使用 `miniprogram/data/` 目录中的本地 JavaScript 数据驱动页面展示：

- `places.js`：景点信息，包括所属区域、坐标、开放时间、门票、建议游览时长和注意事项等
- `foods.js`：美食信息，包括分类、推荐店铺、地址、标签和可选参考坐标等
- `regions.js`：区域信息，用于组织区域攻略及关联景点、美食数据
